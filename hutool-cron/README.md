## 📚Hutool-cron 模块介绍

`Hutool-cron`提供简单定时任务的功能。

-------------------------------------------------------------------------------

## 🛠️包含内容

Hutool的定时任务模块与Linux的Crontab使用上非常类似，通过一个`cron.setting`配置文件，简单调用start()方法即可简单使用。
